# 🏁 Getting Started
To follow along with the NorthWestTechGeek Episode 1 Video, follow these steps:

1. Fork the Repository
Click the Fork button at the top right of this page. This creates a copy of this project under your own GitHub account so you can save your changes.
3. Clone to Your Machine
Open your terminal or VS Code and run:
`git clone https://github.com/YOUR_USERNAME/portfolio-starter-kit.git`
3. Open in VS Code
Open the folder and launch Live Server to see your site update in real-time as you code.

## 🏗️ The Engineering Workflow
In this course, we don't just "save" files. We follow the professional developer pipeline:

Modify: Change your name and brand colors in index.html and style.css.
Stage: `git add .`
Commit: `git commit -m "Personalized my portfolio brand"`
Push: `git push origin main`
Deploy: Connect your GitHub repo to Render for a live URL.
